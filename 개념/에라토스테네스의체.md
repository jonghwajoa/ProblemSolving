n의 소수인지 식별하기 위해서

1부터 n-1 까지 전부 나머지가 0인지 확인 하면 O(n)

어떤수 n = a\*b 라고 생각하면
a 또는 b 는 반드시 루트n보다 작아야함

a < 루트n || b < 루트n
그래야만 a\*b = n 이 성립됨

O(nloglogn) 수준으로 떨어진다고 함

```java
for (int i = 2; i * i <= m; i++) {
			if (isPrime[i]) {
				continue;
			}

			for (int j = i + i; j <= m; j += i) {
				isPrime[j] = true;
			}
        }
```
